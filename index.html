Wikinspire

TNM094 - Medietekniskt kandidatprojekt (Media Technology - Bachelor Project)

A bachelor project in Media Technology where the task were to use Wikipedias existing API and visualize information in a time and place related view on a webpage. The final product was made using HTML, CSS and JavaScript. There were some standard libraries used as three.js.

The instructions we got were the following (in Swedish):

Projekt H (Interaktiv visualisering och navigering av Wikipedia)

Målet med projektet är att utveckla en webbapplikation som visualiserar tid- och plats-information från Wikipedia-artiklar i länkade vyer och tillåter filtrering och navigering av dessa data. På så sätt kan användare se hur artiklar relateras i tid och rymd, t ex vad som hände i Polen samtidigt som Nicolaus Copernicus levde, eller vilka platser som berörs i hans artikel. Systemet kan använda Wikipedias publika API och presentera även ytterligare information, t ex associationsgrafer.
  • Date: Spring 2016
  • Client: Examiner
  • Category: Web development, Visualisation, Data mining

Twilight Sparkle

TNM085 - Modelling Project

Ny länk!

A modelling project where the task were to use physics

These were the instructions from the course page:

This project gives the possibility to use the knowledge acquired from the Modellbygge och Simulering course and put into practice together with programming skills. In this project students work in small groups. Each group is asked to make an animation of a physical system. This implies to determine the mathematical model first and then simulate the system on the basis of the model and represent a graphical animation of the system itself. Each group is free to choose the system to work on and also how to implement the model and the graphical animation. The work of each group is described in a written report. An oral presentation takes place at the end of the project where each group will present their own results.

  • Date: Spring 2016
  • Client: Examiner
  • Category: Physics, Modelling, WebGL, three.js

Lego Database

TNMK30 - Elektronisk Publicering (Electronic Publishing)

Our first web project using an existing database with Legos and Lego sets. We were supposed to create webbased interface where the user could search for different Lego pieces and show the results in some way.

Following instructions is in swedish:

Projektuppgiften går ut på att designa och implementera ett webbaserat gränssnitt för att söka på information i en befintlig databas. Webssidan ska vara komplett med navigering, överblick, sökning och strukturerad presentation av stora mängder data. Databasen som vi föreslår att ni använder är given och förslag på möjliga uppgifter följer. Även egna idéer kan diskuteras om särskilt intresse finns.

  • Date: Fall 2013
  • Client: Examiner
  • Category: Database, PHP, HTML, CSS
